- вёрстка солгасно дизайну

- решение бизнес задачи, согласно требованиям
- выполнять задачу можно любыми инструментами ( фреймворками ) 

`Будет плюсом:`
- использование react
- использование rjxs
- добавление кастомной валидации на поля ввода
- использование анимаций
